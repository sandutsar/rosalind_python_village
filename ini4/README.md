## INI4

### Conditions and Loops

**Problem**

$\textcolor{green}{\textsf{Given:}}$  Two positive integers ***a*** and ***b*** (***a*** < ***b*** < **10000**).

$\textcolor{green}{\textsf{Return:}}$ The sum of all odd integers from ***a*** through ***b***, inclusively.

**Sample Dataset**

```
100 200
```

**Sample Output**

```
7500
```
